# Plan: Generate Flashcard CSV Scripts

## Overview

Create one `__generate_flashcards.py` script per `sources/` folder. Each script reads the
folder's processed CSV and writes a schema-compliant CSV to `spreadsheets/`. No new
dependencies — stdlib only (`csv`, `pathlib`). All scripts are run from the project root
with `.venv` activated.

---

## Schema reminder (from `spreadsheets/schema.md`)

| Column | Meaning |
|---|---|
| `front_text` | Italian word / infinitive shown on card front |
| `front_labels` | `label: value \| ...` context chips |
| `back_highlight` | The "answer" — conjugated form, article+noun, etc. |
| `back_text` | English meaning |
| `audio` | Italian text to synthesise |

---

## Scripts to create

### 1. `sources/1_nouns_by_frequency/__generate_flashcards.py`

- **Input**: `sources/1_nouns_by_frequency/nouns_translated.csv` (`italian`, `english`)
- **Output**: `spreadsheets/nouns.csv`
- **One row per noun**
- **Schema mapping**:
  - `front_text` = Italian noun (e.g. `parte`)
  - `front_labels` = `noun`
  - `back_highlight` = *(empty — no article data available in source)*
  - `back_text` = English (e.g. `part`)
  - `audio` = Italian noun

---

### 2. `sources/2_adjectives_by_frequency/__generate_flashcards.py`

- **Input**: `sources/2_adjectives_by_frequency/adjectives_translated.csv` (`italian`,
  `english`). The `italian` column stores all inflected forms as a single string, e.g.
  `primo/a/i/e`, `grande/i`, `blu`.
- **Output**: `spreadsheets/adjectives.csv`
- **One row per adjective**
- **Schema mapping**:
  - `front_text` = Italian forms string (e.g. `primo/a/i/e`)
  - `front_labels` = `adjective`
  - `back_highlight` = Italian forms string (same — the highlighted "answer" on the back)
  - `back_text` = English (e.g. `first`)
  - `audio` = base form: text before the first `/`, or the full string if no `/`
    (e.g. `primo` from `primo/a/i/e`, `grande` from `grande/i`, `blu` from `blu`)

---

### 3. `sources/3_verbs_infinito_by_frequency/__generate_flashcards.py`

- **Input**: `sources/3_verbs_infinito_by_frequency/verbs_translated.csv` (`italian`,
  `english`)
- **Output**: `spreadsheets/verbs_infinitive.csv`
- **One row per verb**
- **Schema mapping**:
  - `front_text` = Italian infinitive (e.g. `essere`)
  - `front_labels` = `verb | form: infinitive`
  - `back_highlight` = Italian infinitive (the word being confirmed)
  - `back_text` = English (e.g. `to be`)
  - `audio` = Italian infinitive

---

### 4. `sources/4_verbs_presente_by_frequency/__generate_flashcards.py`

- **Input**: `sources/4_verbs_presente_by_frequency/verbs_conjugated.csv` (14 columns)
- **Output**: `spreadsheets/verbs_presente.csv`
- **6 rows per verb** (one per pronoun: io, tu, lui/lei, noi, voi, loro)
- **Schema mapping per pronoun**:
  - `front_text` = Italian infinitive (e.g. `vivere`)
  - `front_labels` = `infinitive: vivere | tense: present | subject: io`
  - `back_highlight` = conjugated form (e.g. `vivo`)
  - `back_text` = English of that form (e.g. `I live / I am living`)
  - `audio` = conjugated form
- Column pairs to iterate: `(io, io_english)`, `(tu, tu_english)`, `(lui/lei,
  lui/lei_english)`, `(noi, noi_english)`, `(voi, voi_english)`, `(loro, loro_english)`
- Skip any row where the conjugated form is empty (handles partial output files)

---

## Shared conventions across all scripts

- Encoding: UTF-8 read and write
- `csv.DictWriter` with `quoting=csv.QUOTE_MINIMAL` (fields with commas/pipes quoted
  automatically)
- `FIELDNAMES = ["front_text", "front_labels", "back_highlight", "back_text", "audio"]`
- `OUTPUT_DIR = PROJECT_ROOT / "spreadsheets"`; `OUTPUT_DIR.mkdir(parents=True, exist_ok=True)`
- Print a summary line: `Written N rows to spreadsheets/X.csv`
- `PROJECT_ROOT = pathlib.Path(__file__).resolve().parents[2]`
